
# Release 2.1.3 (2026-06-24)

## Bug fixes

- **triage 4 new CVEs blocking the 2.1.2 prod publish (#362)**
The 2.1.2 release CD failed at the Trivy publish gate on 4 new CVEs (release + docs jobs and the java images passed; base/python/ruby/go/rust docker-publish failed). None is exploitable in the images' actual runtime usage:

- CVE-2026-39883 (go.opentelemetry.io/otel/sdk, fixed 1.43.0) — OTel SDK transitive in the prebuilt scorecard/trivy binaries; tools do not export telemetry on attacker-controlled paths.
- CVE-2026-55200 (libssh2-1t64, no fix) — git SSH transport to known hosts only; joins the existing libssh2 entry.
- CVE-2026-52908 / CVE-2026-52910 (linux-libc-dev) — kernel-header false positives; containers use the host kernel.
